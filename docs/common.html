前端 UI 组件库

组件风格

要求

色调

#409EFF #67C23A <#F56C6C
#F5F7FA #EBEEF5
#304156 #263445 #000000

页面内引用 variables.scss，调用相应色值即可

例如：$primaryColor、$greenColor

字体

$font-size-smaller（12px）
$font-size-small（13px）
$font-size-medium（14px）
$font-size-large（16px）
$font-size-larger（18px）
$font-size-largest（20px）

如涉及到特殊字体大小可按具体需求定义

默认字体类名如下：

font-family：Helvetica Neue, Helvetica, PingFang SC, Hiragino Sans GB, Microsoft YaHei, Arial, sans-serif

图标

项目引用 Font Awesome 图标库，如果项目有特殊要求可后期加入其他图标

图标已封装成组件

实用例子：< icon icon-class="fa-lock" />

按钮

确认 取消
新建文件 上传 删除
文字按钮

内边距 padding: 10px 20px;

字体大小 font-size: 14px;

圆角 border-radius: 4px;

单选框

选项一 选项二 选项三

选项默认间距：30px

选项与按钮间距：10px

多选框

选项默认间距：30px

选项与对应按钮间距：10px

表单

表单长度 width 可配置，默认100%

行高 line-height: 38px; 内间距 padding: 0 15px;

选中边框颜色为默认主题色：$primaryColor(#409EFF)

禁用输入框背景色值为 #F5F7FA

分页器 - Pagination

已封装成组件，按配置传入相应字段即可

配置字段：total - 总条数；page - 当前页码；limit - 每页条数；pageSizes - 每页显示条目个数

上传组件 - Upload

将文件拖到此处，或点击上传
只能上传jpg/png文件，且不超过500kb

默认宽度 width: 360px; 高度 height: 180px

已封装成组件，按配置传入相应字段即可

配置字段：accept - 上传文件格式，例：['jpg','png']

multiple - 多文件上传

弹窗组件 - FormDialog

打开弹窗

带 form 数据的弹窗已封装为组件 FormDialog，配置字段根据后台返回 JSON 数据动态绑定

根据 JSON 数据中 Layout 字段的值判断类型，例如：'Text' 对应 输入框，'Select' 对应下拉选择器等；具体配置字段可查看组件源码注释

表格组件 - ListTable

弹窗编辑 行内编辑

表格已封装为组件 ListTable，表头字段根据后台返回 JSON 数据动态绑定

编辑模式分为 '弹窗编辑' 和 '行内编辑'，可根据不同场景需求更改 inlineEdit 字段配置， 默认为 '弹窗编辑'

配置字段：actions - 操作按钮，例如：['edit', 'delete']

summary - 是否需要合计数据

summaryField - 合计字段

inlineEdit - 是否支持在表格行内直接编辑

columns - 每列参数

sortable - 是否需要排序

layout - 表单类型

dateFormat - 日期格式

unit - 字段单位，例如：元、kw/h

dateFormat - 日期格式

dateFormat - 日期格式

options - 选择器配置项

showType - 表格内数据显示方式，属于对展现形式有特殊要求的配置项，例如：colorLump 色块显示

富文本编辑器 - Tinymce

富文本编辑器使用 Tinymce，具体文档 点击查看

配置字段：id - 组件ID

value - 组件内容

width - 组件宽度

height - 组件高度

menubar - 配置菜单

toolbar - 工具菜单

准备 生产 取 消 确 定
